Lamellipodia

Lamellipodia are sheet-like cellular structures that consist of a dense actin network an several accessory proteins. Many cell types use this structure to crawl on flat surfaces.

The Filament Based Lamellipodium Model (FBLM)

Collaboration with
   Christian Schmeiser, University of Vienna, Austria
   Dietmar Ölz, NYU, USA
   Nikolaos Sfakianakis, Mainz, Germany
   Stefanie Hirsch, University of Vienna, Austria

Project Description: The FBLM (Filament Based Lamellipodium Model) is a continuous, nonlinear PDE model whose derivation is based on the microscopic description of actin filaments and interacting proteins. It links macroscopic phenomena, such as chemotaxis induced shape changes, cell-cell interactions, etc. to the underlying biological players.
